<<카카오맵 API>>

1. 관광지 목록 csv 파일 load
2. 카카오맵API 요청 - 요청변수 : [관광지명]
3. 응답결과 : [위도, 경도, 주소]

- Google Map API는 제주도 관광지가 표시되지 않은 경우가 있어서 Kakao Map API로 변경하였음

***note: 관광지 목록 파일을 먼저 로드하여 그 관광지 목록에 있는 [관광지명]을 카카오맵 api 요청주소의 파라미터로 넘겨주어 그 관광지의 [위도, 경도, 주소]를 가져왔다.

<<Google Map(Google places) API>>
1. 관광지 목록 csv 파일 load
2. 장소 세부정보 확인을 위한 [장소ID] 가져오기
    1) 요청변수로 장소ID 찾는 Google Map API 요청
        - 요청변수 : [관광지명]
    2) 응답결과 : [장소ID]
3. 장소 세부정보 확인
    1) 장소ID로 세부정보 가져오는 Google Map API 요청
        - 요청변수 : [장소ID]
    2) 응답결과 : [평점, 리뷰개수]

***note: 관광지 목록 파일을 먼저 로드하여 그 관광지 목록에 있는 [관광지명]을 가져온다.
구글 플레이시스에서 최종적으로 [평점, 리뷰개수]를 얻기위해서는 관광지의 [장소ID]를 먼저 가져온 후에 그 [장소ID]로 그 관광지의 장소의 세부정보를 확인할 수 있다.
그래서 먼저 관광지 목록에 있는 [관광지명]을 구글맵 api 요청주소의 파라미터로 넘겨주어 [장소ID]를 찾아온다.
그리고 그 [장소ID]를 장소 세부정보를 확인하는 구글맵 api 요청주소의 파라미터로 넘겨주어서 구글 플레이시스에서 그 관광지에 대한 [평점, 리뷰개수]를 가져온다.
